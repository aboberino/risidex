# risidex

## Présentation

Lors de l’ouverture de l’application, vous pouvez voir un message d’accueil vous présentant l’application et son utilité. 

Sur le menu déroulant à gauche l’onglet “Risidex”  va vous afficher la liste des différents Risimon que vous pourrez trouver à Issou, avec leur nom, leur types ainsi qu’une photo.

L’onglet “Accueil” quand à lui vous ramène sur la page d’accueil.
